## 📌 项目概述
**Grid Survival Game** 是一款基于 C++ 与 Raylib 开发的 2D 动作游戏。在 20x14 的离散网格空间内，系统模拟了主控实体与追踪实体的动态交互。本项目重点攻克了底层寻路算法效率、本地化数据存储逻辑以及跨环境编译链配置等技术挑战。

## 🕹️ 游戏玩法 (Gameplay)

Grid Survival Game 采用简约而紧张的策略生存机制：

* **核心目标**：在有限的网格竞技场内生存并收集资源。你需要在 AI 怪物的围捕下，通过精准的走位避开拦截。
* **AI 寻路机制**：每一个怪物都搭载了 **A* 寻路引擎**，它们会根据你当前的位置实时计算出最优拦截路径，始终向你靠拢。
* **游戏失败判定**：一旦你的角色坐标与任何怪物坐标重合，系统即判定为“生存失败”。
* **排行榜挑战**：生存时间越久、收集的积分越多，你的成绩就会越亮眼。挑战自我，看看你能否凭借策略冲进全班 **Top 5**！

## ✨ 核心技术亮点

| 功能模块 | 技术要点说明 |
| :--- | :--- |
| **A* 寻路算法** | 纯 C 手写实现。集成曼哈顿距离启发函数，实现追踪 AI 对主控实体的实时最优路径计算。 |
| **持久化存储** | 封装高效文件 I/O 接口，实现排行榜数据的 CRUD 操作，支持自动化排序与长效存储。 |
| **环境解耦** | 自研 `compile.bat` 自动化构建脚本，通过相对路径管理依赖，实现无 IDE 环境下的一键秒级编译。 |
| **UTF-8 渲染** | 基于 `LoadFontEx` 与自定义字形提取技术，深度优化中文字库加载，规避字符乱码问题。 |
